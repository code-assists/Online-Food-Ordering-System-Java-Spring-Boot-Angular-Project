# 🍔 Online Food Ordering System | Java Spring Boot, Angular & MySQL Project  

**Online Food Ordering System** is a **full-stack web application** developed using **Java Spring Boot, Angular, and MySQL**.  
This project manages **Food Items, Categories, Companies, Customers, Orders, and Sales**.  

The system supports **two types of users**:  
- **Admin** → Manage customers, food items, categories, orders, and reports.  
- **Customer** → Register, log in, browse food items, add to cart, make payments, place orders, and give feedback.  

This is a **major final year project** for students, complete with **source code, MySQL database, and documentation**.

## 📖 Abstract  

The **Online Food Ordering System** simplifies the process of ordering food online.  
- **Admins** → Manage food items, categories, customer records, and orders.  
- **Customers** → Can register, order food items, track orders, make payments, and submit feedback.  

This project demonstrates **full-stack development using Angular (frontend) + Spring Boot (backend) + MySQL (database)** and is highly relevant for **final year Computer Science students**.  

---

## 📂 Modules & Features  

### 👤 Customer Module  
- Admin can add, edit, update, and delete customers.  
- Customers can register, log in, update their profile.  
- CRUD operations with role-based access restrictions.  

### 🍕 Food Items Module  
- Admin can manage (CRUD) food items.  
- Customers can browse and view food items.  

### 🥗 Food Item Category Module  
- Admin can manage food categories (CRUD).  
- Customers can view food item categories.  

### 🛒 Order Module  
- Admin manages customer orders (CRUD).  
- Customers can view their order history and receipts.
